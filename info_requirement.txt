# Информация о планетах:
# Техническая информация:
# * Название планеты (name)
# * Изображение планеты на макете (image)
# * Орбита планеты на макете (orbit)
# Орбитальные характеристики:
# * Эксцентристет орбиты (orbit_excenter)
# * Продолжительность оборота, текстом в земных годах или днях (orbit_year)
# * Орбитальная скорость, км/с (orbit_speed)
# * Перигелий (ближайшее расстояние до Солнца), а.е. (orbit_perihelion) 
# * Афелий (наибольшее отдаление от Солнца), а.е. (orbit_aphelion)
# * Большая полуось, а.е. (вычисляется как (orbit_perihelion + orbit_aphelion) / 2)
# Физические характеристики:
# * Мантисса массы, кг. (mass_mantissa)
# * Десятичный показатель меры массы (mass_exp)
# * Мантисса объёма, км^3 (volume_mantissa)
# * Десятичный показатель меры объёма (volume_exp)
# * Средняя плотность, г./см^3 (вычисляется как mass_mantissa / volume_mantissa * 10 ** (mass_exp - volume_exp + 3 - 5))
# * Средний радиус, км (radius)
# * Площадь поверхности, км^2 текстом (area)
# * Наклон оси, в градусах, (tilt)
# * Ускорение свободного падения, (gravity)
# * Космические скорости, кортеж, км/с (space_speed)
# * Продолжительность суток, кортежом в часах, минутах, секундах (sidereal_period)
# Температура, кортежем чисел: мин., ср. и макс., в кельвинах (temp)
# Состав атмосферы, словарь {"элемент": процент} (atmo)
